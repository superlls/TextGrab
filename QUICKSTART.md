# TextGrab 快速使用指南

## 启动应用

```bash
./textgrab
```

应用会在菜单栏显示 "TextGrab"，不会打开窗口。

## 使用方法

### 方式 1: 使用快捷键（推荐）

1. 按下 **⌘⇧W** (Command + Shift + W)
2. 用鼠标框选要识别的屏幕区域
3. 松开鼠标后自动识别
4. 识别完成后文本自动复制到剪切板
5. 按 **⌘V** 即可粘贴

### 方式 2: 使用菜单栏

1. 点击菜单栏的 "TextGrab" 图标
2. 选择 "截图识别 (⌘⇧W)"
3. 用鼠标框选要识别的区域
4. 等待识别完成

## 首次运行权限设置

### 必需权限

**屏幕录制权限**
1. 打开 系统偏好设置 → 安全性与隐私 → 隐私 → 屏幕录制
2. 点击左下角锁图标解锁
3. 勾选 "Terminal" 或 "iTerm"（取决于你使用的终端）
4. 重启 textgrab

### 可选权限（用于全局快捷键）

**辅助功能权限**
1. 打开 系统偏好设置 → 安全性与隐私 → 隐私 → 辅助功能
2. 点击左下角锁图标解锁
3. 勾选 "Terminal" 或 "iTerm"
4. 重启 textgrab

## 使用技巧

- 按 **ESC** 可以取消截图
- 识别完成后会有系统通知
- 程序会一直运行在后台，可以重复使用
- 点击菜单栏图标选择"退出"可以关闭程序

## 常见问题

### 快捷键不工作
- 检查是否授予了辅助功能权限
- 检查是否有其他应用占用了 ⌘⇧W 快捷键

### 截图失败
- 检查是否授予了屏幕录制权限
- 修改权限后需要重启应用

### 识别不准确
- 确保截图清晰
- 避免截取过小或模糊的文字
- 尽量选择纯文本区域

## 开机自启动（可选）

1. 系统偏好设置 → 用户与群组 → 登录项
2. 点击 "+" 添加 textgrab
3. 勾选"隐藏"选项

## 退出应用

点击菜单栏的 TextGrab 图标，选择"退出"。
